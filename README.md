# Calculadora simples em Python (Dois Números)

Projeto simples de uma calculadora feita em Python utilizando execução via terminal.

---

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão com validação de erro (divisão por zero)
- Menu interativo em loop
- Opção de saída do programa

---

## Lógica do sistema

O programa funciona através de um menu interativo no terminal, onde o usuário escolhe a operação desejada.

Após a seleção:
- O sistema solicita dois números
- Realiza o cálculo correspondente
- Exibe o resultado na tela

O programa permanece em execução até o usuário escolher a opção de sair.

---

## Tratamento de erros

- Impede divisão por zero
- Valida opções inválidas no menu

---

## Tecnologias utilizadas

- Python (3.14.4)
